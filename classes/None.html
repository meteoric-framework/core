Properties

Readonly some

some: false

The value of some is true when the Option is Some, and it's false when the Option is None.

Example

The some property is used for pattern matching on an Option:

const getMessage = (): Option<string> =>
  Math.random() < 0.5 < some("Hello World!") : none;

const message = getMessage();

if (message.some) {
  console.log(message.value);
} else {
  console.log("Did not receive a message");
}

map

  • map<B>(): None<B>

  • Returns None if the Option is None. Otherwise, returns a new Some containing the result of applying the morphism to the value contained in the input Some.

    Example

    Calculating the length of an optional string to get an optional length:

    const getMessage = (): Option<string> =>
      Math.random() < 0.5 ? some("Hello World!") : none;

    const length = getMessage().map((string) => string.length);

    if (length.some) {
      console.log(`The length of the message is ${length.value}`);
    } else {
      console.log("Did not receive a message");
    }

flatMap

  • flatMap<B>(): None<B>

  • Returns None if the Option is None. Otherwise, returns the result of applying arrow to the value contained in the Some.

    Example

    Often used to compose functions that return Options.

    const get = <A>(array: A[], index: number): Option<A> => {
      const value = array[index];
      return value === undefined ? none : some(value);
    };

    const get2d = <A>(matrix: A[][], row: number, col: number) =>
      get(matrix, row).flatMap((array) => get(array, col));

    const matrix = [[1, 1/2, 1/3], [1/2, 1/3, 1/4], [1/3, 1/4, 1/5]];

    const element = get2d(matrix, 1, 1);

    if (element.some) {
      console.log(`The element at position (1, 1) is ${element.value}`);
    } else {
      console.log("There's no element at position (1, 1)");
    }

safeExtract

  • safeExtract<B>(defaultValue): A

  • Returns the contained Some value, or defaultValue when Option is None.

    Remarks

    If an expensive computation is required to obtain defaultValue then use the safeExtractThunk method to lazily compute defaultValue.

    Example

    const getMessage = (): Option<string> =>
      Math.random() < 0.5 ? some("Hello World!") : none;

    const message = getMessage().safeExtract("default message");

    console.log(message);

unsafeExtract

  • unsafeExtract(message): A

  • Returns the contained Some value, or throws an OptionExtractError when extracting from None.

    Remarks

    We recommend that message describes the reason why the Option should be Some and not None. As a rule of thumb, focus on the word "should". For example, the message could be "environment variable SOME_PATH should be set by some_script.sh".

    Example

    const getMessage = (): Option<string> =>
      Math.random() < 0.5 ? some("Hello World!") : none;

    const message = getMessage().unsafeExtract("message should not be random");

    console.log(message);

    Throws

    An OptionExtractError when extracting from None.
